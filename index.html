How MCP Server Works

Transparent privacy protection in your AI workflow

1

Your Prompt

Send a prompt containing sensitive data to your AI tool

2

MCP Intercepts

PII is detected and replaced with reversible tokens

3

AI Processes

Sanitized prompt goes to the model, response comes back

4

Data Restored

Tokens are replaced with original values in the response

Quick Start

Get protected in under a minute

JSON 
{
  "mcpServers": {
    "anonymize": {
      "command": "npx",
      "args": ["-y", "@anonym-legal/mcp-server"],
      "env": {
        "ANONYM_API_KEY": "your-api-key"
      }
    }
  }
}

Add this to your claude_desktop_config.json or Cursor settings
